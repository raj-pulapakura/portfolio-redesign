# Swiss Precision --- Portfolio Art Direction

## Vision

A structured and rational design inspired by Swiss International Style.

The portfolio emphasizes clarity, alignment, and typographic precision.

## Core Principles

-   Grid-first design
-   Functional aesthetics
-   Neutral typography
-   Systematic spacing

## Typography

Primary: - Neue Haas Grotesk - Inter - Suisse Intl

Hierarchy: - Hero: 56--72px bold - Section headings: 28--32px - Body:
16px

Project numbering emphasized: 01 02 03

## Color System

Minimal palette.

-   Background: #FFFFFF
-   Text: #111111
-   Accent: #0057FF

Accent used sparingly for: - links - active states - highlights

## Layout

Strict grid alignment.

-   12 column grid
-   consistent spacing scale
-   8px spacing system

Sections often left-aligned.

Projects appear in structured lists.

Example:

01 Project Name\
02 Project Name\
03 Project Name

## Motion

Functional and subtle.

-   quick hover highlights
-   simple underline transitions
-   page transitions 200--300ms

No decorative motion.

## Interaction

Clear affordances.

-   Underline on hover
-   Predictable navigation
-   Consistent interaction patterns

## Project Presentation

Case studies structured for clarity.

Example:

Project\
Role\
Year\
Technologies

Followed by:

Problem\
Approach\
Result

## Emotional Tone

Precise, rational, professional.
